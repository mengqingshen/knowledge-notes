## 技巧

### Q：编译遇到 startup lock

```bash
$ flutter doctor
Waiting for another flutter command to release the startup lock...
```

A：删除 lockfile 文件

```bash
$ rm ~/Programs/flutter/bin/cache/lockfile
```
